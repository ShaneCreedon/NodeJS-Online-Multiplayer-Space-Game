Online Multiplayer Space Game - Built by Shane

Project NX

Instructions

    1. Accelerate your ship forward with the Up arrow key.

    2. Rotate your ship around with the Left & Right arrow keys.

    3. Engage boost thrusters with the shift key for rapid acceleration

    4. Fire laser projectiles at other players with the 'X' key

    5. Chat with other players using the Enter key

About the Project

    Author: Shane

    Summary: This project is a 2D online multiplayer space game playable via the web.

    Tech Stack: NodeJs, Socket.io, Express, Heroku, and of course, HTML, CSS & Javascript.
